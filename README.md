# 严峫 AI 情感伴侣

一个基于 Next.js 的角色化聊天应用，将人物设定为《破云》中的严峫，让他以阳光正义、深情可靠的形象与你建立情感连接。

## 特性

- 以严峫为角色的情感陪伴聊天界面
- 漫画风视觉氛围与高级暗色 UI
- 关键词触发的角色化回应逻辑
- 可扩展为真实 AI 服务的后端接口

## 开始使用

### 先决条件

- Node.js（18 或更高版本）
- npm 或 yarn

### 安装

```bash
npm install
```

### 运行

```bash
npm run dev
```

打开浏览器访问 `http://localhost:3000`。

## 使用说明

- 在输入框中向严峫说出你的心事。
- 严峫会以“阳光正义、深情守护”的语气回复你。
- 你可以问他：“我今天很累”，“你会一直守护我吗”，或“你怎么看我的选择”。

## 自定义

- 若要接入真实 AI 服务，请修改 `app/api/chat/route.ts`。
- 可以将当前的角色化逻辑替换为对接 OpenAI 或其它对话模型。

## 构建

```bash
npm run build
npm start
```

## 说明

该项目目前只使用内置角色化回复逻辑，适合做严峫形象的情感对话演示。
